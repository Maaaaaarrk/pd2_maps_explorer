================================================================
  PD2 Tier 1 Map Comparison — Season 12 vs Season 13
  Average base XP per monster across the Hell spawn pool
================================================================

Method:
  Base XP per monster (Hell) = MonLvl.XP(H)[level] * MonStats.Exp(H) / 100
  Per-map value              = mean across the map's nmon Hell spawn pool
  Tier average               = mean across the maps in the tier
  All XP values use CURRENT monster stats (S13 MonStats.txt). This
  isolates the effect of the tier roster change; underlying monster
  Exp% changes between seasons are not captured.
