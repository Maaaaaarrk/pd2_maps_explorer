# Monster Resistance Survey by Map

Highest Hell-difficulty resistance value across the regular spawn pool in each map (monsters referenced via `nmon/mon/umon` in `Levels.txt`). Hardcoded event/quest bosses are **not** included because their map-association can't be inferred reliably from naming. Poison resistance values of `1000` (hardcoded "fully immune" placeholders) are excluded so the listed poison res reflects what normal spawns actually carry.

Values ≥ 100 mean the monster is **immune** to that element.
